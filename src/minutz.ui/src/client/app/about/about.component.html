Features

  • Ready to go, statically typed build system using Gulp for working with TypeScript.
  • Production and development builds.
  • Development server with live reload.
  • TypeScript definition management using Types.
  • Basic Service Worker, which implements "Cache then network strategy".
  • Token authentication using OpsAuth.
  • Http Interceptor to append tokens to each request.

OAuth

  • OAuth has been configured for development purposes only
  • You will need to contact security to get a client_id and client_secret.
  • You will to configure octopus variables (prod.ts) for deployment.
