Current features

  • Loading indicator.
  • Unauthorized page (401).
  • Not found page (404).

Loading indicator

This is a simple loading indicator. Allow that is needed to make this work is:

  • Create a variable inside your *.component.ts called loading of type boolean like so public loading:boolean = false;
  • Inside your *.component.html add *ngIf="!loading" for content that will display BEFORE loading needs to happen and then *ngIf="loading" for the loading section with needs the html tag sd-loading
  • Switch the loading icon to true when the loading icon must be displayed and false to hide it and show the page content

Unauthorized Page (401)

This is a simple unauthorized page that you application can redirect users to if they try perform an unauthorized action.

Not Found Page (404)

This is a simple not found page that your application can redirect any request to, or redirects any request to that does not have a route within the application. This is achieved by adding a second route to the routing module called path: '**'
